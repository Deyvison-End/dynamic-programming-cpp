# Recursão, Memoization e Programação Dinâmica em C++

Projeto desenvolvido durante os estudos de Análise de Algoritmos e Programação Dinâmica utilizando diferentes estratégias para resolução de problemas clássicos.

## Problemas Implementados

### Maximum Subsequence Sum

Determina a subsequência contígua de maior soma em um vetor.

### Subset Sum

Verifica se existe um subconjunto de elementos capaz de atingir uma soma específica.

* Programação Dinâmica

* Solução com Memoization

### Subset Sum
